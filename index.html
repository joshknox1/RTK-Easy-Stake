        Position source

        For RTK-grade accuracy, pair the Facet with your phone as a Bluetooth GPS (mock-location) source using an app like "Bluetooth GPS" or "GPS Connector" — then this page's normal phone GPS reading will already be RTK-corrected.

        Attempts a direct Web Bluetooth link to read NMEA and show true fix quality (RTK Fixed / RTK Float) instead of relying on the phone's GPS chip. Requires Chrome on Android and the Facet's Bluetooth Protocol set to BLE or Dual in its system menu. If it can't find the right service, it will fail harmlessly — phone-GPS mode keeps working either way. See README for details and how to supply a custom service/characteristic UUID if your firmware differs.
